# Project Website created with Node.js

## How run project:
- 1 step run: 
```
npm or yarn install
```
This command will install node_modules

- 2 step run: 
```
nodemon app.js
```

Use `nodemon app.js` run the project.

## Packages used:
- **Frontend:**
  - HTML5
  - CSS3
  - Javascript
  - Bootstrap
- **Backend:**
  - NodeJS
  - Nodemon
  - Mysql2
  - Sequelize
  - Express
  - Express-handlebars
