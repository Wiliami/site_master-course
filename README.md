# Website created with Node.js

## How run project:
- 1 step run: Install all dependencies
```
npm install
```
- 2 step run: 
```
nodemon app.js
```
Use `nodemon app.js` run the project.

## Tools used in the project:
- **Frontend:**
  - JQuery
  - Bootstrap
  - Express-handlebars
- **Backend:**
  - NodeJS
  - Express
  - Firebase
